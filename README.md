# AllInOneTools Documentation

AllInOneTools is a privacy-first platform offering free online tools
with no tracking, no ads, and no sign-ups.

## Mission
Build simple, fast, and trustworthy tools that work directly in the browser.

## Philosophy
- No data collection
- No user accounts
- No dark patterns
- Performance over hype

## Transparency
This repository exists to document values, decisions, and long-term vision.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
